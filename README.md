# MindolifeTest – QA Automation (Python + Selenium)

Automated UI tests for a web application using Python + Selenium.
Includes basic structure and examples for running tests locally.

## Tech Stack
- Python
- Selenium
- (Optional) pytest / unittest
- WebDriver (ChromeDriver / GeckoDriver)

## What’s Covered
- Smoke tests (critical user flows)
- Regression scenarios
- Basic validations (UI elements, navigation, forms)
